## 技术背景
手眼协调是机器人视觉与控制结合的关键技术，通过建立机器人末端执行器与相机之间的空间关系，实现基于视觉的精确操作。典型应用包括：

- 视觉引导抓取
- 装配操作
- 视觉检测与质量控制
- 自动化生产线

## 常用工具和库
- OpenCV - 计算机视觉库
- ROS/ROS2 - 机器人操作系统
- MoveIt - 机器人运动规划框架
- PCL - 点云处理库

- 计算平台（支持实时图像处理）
