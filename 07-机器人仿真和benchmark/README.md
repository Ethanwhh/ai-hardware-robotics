## 学习路径
1. 首先阅读 [仿真配置文档](01仿真配置文档.md) 了解基础配置
2. 根据需要选择 ManiSkill 或 Isaac Sim 进行环境配置
3. 参考详细文档学习如何使用仿真环境
4. 尝试特定场景，如家务机器人环境配置

## 环境要求
- Python 3.8+
- CUDA支持（用于高质量渲染）
- MuJoCo物理引擎
- 显卡要求：建议NVIDIA GTX 1080或更高
